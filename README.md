# Calculadora IMC Vue

Projeto simples feito com Vue.js para calcular o IMC de usuários.

## Funcionalidades

- Cadastro de nome, altura e peso
- Cálculo automático do IMC
- Classificação do resultado
- Salvamento dos dados no `localStorage`
- Tela de histórico dos usuários cadastrados
- Navegação entre páginas com Vue Router
- Interface com botões animados e background dinâmico

## Tecnologias

- Vue.js
- Vue Router
- Vite
- CSS
- localStorage
